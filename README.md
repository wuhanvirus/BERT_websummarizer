# BERT_websummarizer      

## 사용법
<br/>
<br/>
## KOREAN
<br/>
<br/>
디렉토리내에서

**nodemon npm start**    (80 포트 개방)

이후

**python fapi.py**    (요약모델을 api로 띄울 파이썬 코드)

---------------------------------------
<br/>
<br/>
## ENGLISH
<br/>
<br/>
Within the directory

**nodemon npm start** (80 ports open)

after

**python fapi.py** (Python code to load the summary model as api)
